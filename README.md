# QRForge 🚀

QRForge is a modern and responsive QR Code Generator built with React, Vite, TypeScript, and Tailwind CSS. It allows users to generate customizable QR codes with advanced features such as logo embedding, color customization, multiple download formats, dark mode, and QR history.

## 🌟 Features

### QR Code Generation

* Generate QR codes from URLs or text
* Real-time QR preview
* URL validation
* Character counter

### Customization

* Custom QR color
* Custom background color
* Adjustable QR size
* Logo upload and embedding
* Dark / Light mode

### Download Options

* Download QR as PNG
* Download QR as JPG
* Download QR as SVG

### User Experience

* Copy QR content to clipboard
* Responsive design
* Smooth animations
* Toast notifications
* Mobile-friendly interface

### Additional Features

* QR History (stored locally)
* Analytics Information Page
* QR Best Practices Guide
* Modern UI with glassmorphism effects

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Swejal-13/qr-generator.git
```

### Navigate to Project Directory

```bash
cd qr-generator
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
