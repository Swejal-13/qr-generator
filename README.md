# QRForge 🚀

A modern QR Code Generator built with React, Vite, TypeScript, and Tailwind CSS.

Generate beautiful QR codes instantly with custom colors, logo embedding, multiple download formats, dark mode support, and a responsive user experience.

## 🌐 Live Demo

Add your deployed Vercel URL here:

https://your-vercel-url.vercel.app

---

## ✨ Features

### QR Code Generation

* Generate QR codes from text or URLs
* Real-time QR preview
* URL validation
* Character counter

### Customization

* Custom QR foreground color
* Custom QR background color
* Adjustable QR size
* Logo upload and embedding

### Downloads

* Download as PNG
* Download as SVG
* Download as JPG

### User Experience

* Dark / Light Mode
* Responsive design
* Toast notifications
* Copy to clipboard
* Smooth animations
* Mobile-friendly layout

### Additional Features

* QR History (stored locally)
* QR Analytics Information Page
* QR Best Practices Guide
* Built for Digital Heroes

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/qrforge.git
```

Navigate into the project:

```bash
cd qrforge
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
